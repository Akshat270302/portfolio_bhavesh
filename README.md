# Bhavesh Portfolio 2026

This is a production-ready portfolio site built with Next.js (App Router), TypeScript, Tailwind CSS, Framer Motion, and Lucide React icons. The site is data-driven from `data/portfolio.ts` and uses local image assets from the `images/` folder.

Quick start

```bash
npm install
npm run dev
# Open http://localhost:3000
```

Build

```bash
npm run build
npm run start
```

Notes

- Resume is served at `/api/resume` (streams the local `Resume (13).pdf`).
- Editable content and links live in `data/portfolio.ts`.
- Project images are loaded from the `images/` folder at runtime.
